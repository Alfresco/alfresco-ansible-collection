alfresco.platform.utils role – Download artifact with checksum verification

Note

This role is part of the alfresco.platform collection (version 0.2.1).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install alfresco.platform.

To use it in a playbook, specify: alfresco.platform.utils.

Entry point main – Download artifact with checksum verification

Synopsis

  • Downloads an artifact from a remote URL and verifies checksum file.

Parameters

Parameter

Comments

artifact

dictionary / required

Artifact download configuration.

checksum_type

string

Checksum algorithm used to verify the artifact

Choices:

  • "sha1" ← (default)

  • "sha256"

  • "sha512"

dest

string

Destination path where the artifact will be saved.

password

string

Password for authentication

url

string / required

URL of the artifact to download.

username

string

Username for authentication
